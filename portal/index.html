How to Learn

📚 The Learning Workflow

Our materials are architectural guides, not an encyclopedia. Each module provides:

  • PDF Materials: High-level conceptual introductions.
  • AI-First Study: Use Gemini/ChatGPT for deep dives.
  • Theory Notebooks: Reference code and solved logic.
  • Practice Notebooks: Hands-on labs (Colab/Kaggle).

🕒 Time Commitment

9-10 Hours Per Module.

This is a marathon. Take structured breaks to maintain rigor.

💡 The Rigorous Practice Rule

Default notebooks are never enough for mastery.

"Create a new notebook, prompt an AI to generate a custom problem sheet for today's topic, and solve it until the logic is muscle memory."

📂 Real-World Practice

Mimic industry data science workflows.

"Avoid practicing with hardcoded arrays. Instead, work directly with .csv files. Ask your AI agent to generate suitable .csv datasets for the topic you are learning."
