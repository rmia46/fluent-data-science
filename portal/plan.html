9-Module Plan

The ultimate strategy for competitive data science contests.

Duration

9 Modules

9 Hours

Session Split

Theory / Lab

Target

Tabular ML

Module 01: Mathematics for ML

Session 1: Linear Algebra

Learn: Vectors, matrices, dot products, and dimensions. Eigenvalues and PCA foundations.

Practice: Implement matrix operations using NumPy and solve linear systems.

Session 2: Calculus

Learn: Derivatives, partials, and the chain rule. Gradient Descent intuition and optimization.

Practice: Write a gradient descent loop from scratch to minimize cost functions.

Session 3: Probability

Learn: Distributions, Bayes' Theorem, and Z-scores. Standardizing features for model stability.

Practice: Simulate distributions and write functions to apply Bayes' Theorem.

Module 02: Tabular Foundations

Session 1: Pandas & Vectorization

Learn: Fast data loading, indexing, slicing, and avoiding slow loops using vectorized operations.

Practice: 15 unique data manipulation and multi-index challenges.

Session 2: Merging & Aggs

Learn: Mastering .groupby(), .merge(), .join(), and constructing dynamic summary tables.

Practice: Combine multiple relational tables and build custom cross-feature statistics.

Session 3: Data Cleaning

Learn: Imputation strategies (mean/median/mode) vs. tracking missingness with explicit indicator flags.

Practice: Build a robust processing function for heavily corrupted datasets.

Module 09: 4-Hour Mock Competition

The Final Sprint: Apply every skill to a raw, messy dataset in a simulated competition environment. Pick a historic Kaggle tabular problem, use your Day 7 template, and iterate your way to the top under a ticking clock.
