9-Module Plan

The strategy for winning competitive data science contests.

Duration

9 Modules

Daily Rigor

9 Hours

Session Split

Theory / Lab

Target

Tabular ML

Module 01: Mathematics for ML

1. Linear Algebra

Learn: Vectors, matrices, dot products, dimensions. Eigenvalues and PCA foundations.

Practice: Implement matrix operations using NumPy and solve linear systems.

2. Calculus

Learn: Derivatives, partials, chain rule. Gradient Descent intuition and optimization.

Practice: Write a gradient descent loop from scratch to minimize cost functions.

3. Probability

Learn: Distributions, Bayes' Theorem, Z-scores. Standardizing features for model stability.

Practice: Simulate distributions and write functions to apply Bayes' Theorem.

Module 02: Tabular Foundations

1. Pandas Fast

Learn: Fast data loading, indexing, slicing, and avoiding slow loops using vectorized operations.

Practice: 15 unique data manipulation and multi-index challenges.

2. Relational

Learn: Mastering .groupby(), .merge(), .join(), and constructing dynamic summary tables.

Practice: Combine multiple relational tables and build custom cross-feature statistics.

3. Cleaning

Learn: Imputation strategies (mean/median/mode) vs. missingness indicator flags.

Practice: Build a robust processing function for heavily corrupted datasets.

Module 03: Feature Engineering I

1. Categorical

Learn: OHE vs. Label vs. Frequency Encoding. Managing column cardinality and sparse matrices.

Practice: Process high-cardinality values without memory leaks or data leakage.

2. Numerical

Learn: Handling skewness, log transforms, Standard Scaling, and MinMax Scaling choices.

Practice: Identify skewed columns and apply transformations to normalize distributions.

3. Interactions

Learn: Mathematical combinations (ratios, products) and segmenting metrics into ordinal bins.

Practice: Write functions to construct high-impact interaction features on real data.

Module 04: Feature Engineering II

1. Target Encoding

Learn: Encoding categories based on target values. Managing risk with smoothing formulas.

Practice: Manually build a smoothed Target Encoder with out-of-fold protection.

2. Contextual

Learn: Utilizing .transform() in Pandas to create localized peer-group context metrics.

Practice: Build 10 contextual features on a messy transactional dataset.

3. Selection

Learn: Dropping collinear features, variance thresholds, and evaluating importance scores.

Practice: Prune low-performing features from a wide dataset ruthlessly.

Module 05: Validation

1. CV Frameworks

Learn: K-Fold vs. Stratified vs. Time-Series Splits. Avoiding the public leaderboard trap.

Practice: Write a robust Stratified 5-Fold cross-validation loop from scratch.

2. Baselines

Learn: Scikit-Learn API consistency. Training rapid reference baselines (RF, Logistic).

Practice: Plug a baseline classifier into your CV loop and generate an OOF track.

3. Leakage Audit

Learn: Common leakage traps: scaling before splitting and encoding outside folds.

Practice: Audit a broken notebook and refactor it to fix data leakage flaws.

Module 06: Tree Models

1. GBDTs

Learn: GBDT mechanics, leaf-wise vs level-wise growth, and speed optimizations.

Practice: Implement LGBM and XGBoost inside your CV framework side-by-side.

2. Overfitting

Learn: Structural constraints (depth, leaves) vs. stochastic adjustments (subsample).

Practice: Iterate hyperparameters to visually analyze training vs. validation divergence.

3. Taming Trees

Learn: Regularization strategies (L1/L2) and categorical parameters.

Practice: Apply stochastic regularization to bring validation scores into optimal bounds.

Module 07: Tuning & Ensembling

1. Optuna

Learn: Bayesian search optimization and constructing objective functions.

Practice: Set up an automated 50-trial Optuna study for your LightGBM model.

2. Blending

Learn: Ensembling principles and combining low-correlation model structures.

Practice: Extract OOF probabilities and compute a mathematically weighted blend.

3. Post-Processing

Learn: Optimal cutoff boundaries instead of standard 0.5 rules for F1 optimization.

Practice: Construct a custom threshold scanner (0.01 to 0.99) for metric maximization.

Module 08: Pipelines

1. Modular Code

Learn: Architecting clean pipelines with modular Python scripts (Config, Prep, Model).

Practice: Convert notebooks into clean, modular, and versioned executable scripts.

2. Collaborative Git

Learn: Managing branches, PRs, and merge conflicts under competition stress.

Practice: Simulate a live sprint: Team Member A Prep; Team Member B Ensembles.

3. Template

Learn: Consolidating your "Secret Weapon" ready-to-run template notebook.

Practice: Lock in a pipeline that reads CSVs and exports sub files in under 30 minutes.

Module 09: Mock Competition

The Final Sprint: Apply every skill to a raw, messy dataset in a simulated contest. Pick a historic Kaggle problem, use your Day 7 template, and iterate your way to the top.
