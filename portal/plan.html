Learning Path & Curriculum

The 9-Day Hyper-Drive Strategy for Competitive Data Science

Duration

9 Days Total

Daily Rigor

9 Hours

Split

1hr Read / 2hr Practice

Focus

End-to-End Tabular

Day 01: Mathematics for Machine Learning

Session 1: Linear Algebra

Lecture Vectors, matrices, dot products, and dimensions.

Practice Implement matrix operations using NumPy.

Session 2: Calculus

Lecture Derivatives, partials, and the chain rule.

Practice Write a gradient descent loop from scratch.

Session 3: Probability

Lecture Distributions, Bayes' Theorem, and Z-scores.

Practice Simulate distributions and calculate Z-scores.

Day 02: Tabular Foundations

Session 1: Pandas & Vectorization

Lecture Fast data loading, indexing, and avoiding slow loops.

Practice 15 unique manipulation and multi-index challenges.

Session 2: Merging & Aggregations

Lecture Mastering .groupby(), .merge(), and .join().

Practice Combine relational tables and build custom aggregates.

Session 3: Data Cleaning

Lecture Imputation strategies vs. missingness indicator flags.

Practice Build a robust processing function for corrupted data.

Day 03: Feature Engineering I

Session 1: Categorical Encoding

Lecture OHE vs. Label vs. Frequency Encoding. Cardinality management.

Practice Process high-cardinality values without memory leaks.

Session 2: Numerical Scaling

Lecture Skewness, log transforms, and Scaling choices (Z-score, MinMax).

Practice Apply transformations and map distributions with Seaborn.

Session 3: Interaction Features

Lecture Mathematical combinations (ratios, products) and binning.

Practice Construct 5 high-impact interaction features on Titanic data.

Day 04: Feature Engineering II

Session 1: Target Encoding

Lecture Encoding categories based on target values with smoothing.

Practice Build a Target Encoder with OOF protection from scratch.

Session 2: Contextual Transforms

Lecture Using .transform() to create peer-group contextual metrics.

Practice Build 10 contextual features on messy transactional data.

Session 3: Feature Selection

Lecture Dropping collinear features and evaluating importance scores.

Practice Prune the bottom 50% of low-performing features ruthlessly.

Day 05: Validation Architecture

Session 1: CV Frameworks

Lecture K-Fold vs. Stratified vs. Time-Series Splits.

Practice Write a robust, reusable Stratified 5-Fold CV loop.

Session 2: Baseline Models

Lecture API consistency. Training Ridge, RF, and Logistic baselines.

Practice Generate an end-to-end output submission file.

Session 3: Leakage Auditing

Lecture Detecting silent performance-inflation traps.

Practice Audit and refactor a broken, leaky pipeline.

Day 06: Tree-Based Models

Session 1: LightGBM

Lecture Leaf-wise growth and missing values handling.

Practice Implement LGBM inside your OOF CV loop.

Session 2: XGBoost

Lecture Level-wise growth and L1/L2 regularization.

Practice Deploy XGBoost side-by-side with LGBM on same folds.

Session 3: Hyperparameter Deep-Dive

Lecture Structural vs. Stochastic adjustments (Subsample, Depth).

Practice Map overfitting curves to find optimal tree complexity.

Day 07: Optimization & Ensembling

Session 1: Optuna

Lecture Bayesian search optimization for parameter tuning.

Practice Automated Optuna script for 50-trial LGBM study.

Session 2: Blending

Lecture Principles of model ensembling and prediction smoothing.

Practice Compute weighted blends between LGBM and XGBoost.

Session 3: Metric Optimization

Lecture Probability cutoff points vs. standard 0.5 rules.

Practice Threshold scanning loop (0.01 to 0.99) for F1-score maximization.

Day 08: Production Pipelines

Session 1: Modular Scripts

Lecture Architecting clean file paths (data_prep.py, models.py).

Practice Convert notebooks into modular executable scripts.

Session 2: Git Workflows

Lecture Managing branches, PRs, and merge conflicts.

Practice Simulate a live team sprint: Prep vs Ensembling.

Session 3: Master Template

Lecture Building your 30-minute "Ready-to-Run" pipeline.

Practice Lock in a template for instant datathon deployment.

Day 09: Mock Competition

The Final Sprint: Apply every skill to a raw, messy dataset in a 4-hour simulated competition environment.
