Section 1: The 8-Day Hyper-Drive Plan

Day 1: Mathematics for Machine Learning & Data Science

Session 1: Linear Algebra Foundations

Lecture

Vectors, matrices, dot products, and matrix multiplication. Understanding dimensions and shapes in datasets.

Practice

Implement matrix operations using NumPy and solve a basic system of linear equations.

Session 2: Calculus & Gradient Descent

Lecture

Derivatives, partial derivatives, and the chain rule. Intuition behind Gradient Descent optimization.

Practice

Write a simple gradient descent loop from scratch to minimize a basic cost function.

Session 3: Probability & Statistics

Lecture

Distributions (Normal, Binomial), expectation, variance, and Bayes' Theorem. Hypothesis testing intuition.

Practice

Simulate distributions, calculate z-scores, and write a function to apply Bayes' theorem to a medical dataset.

Day 2: Tabular Foundations & Fast Data Manipulation

Session 1: Pandas Essentials & Vectorization

Lecture

Data loading, indexing, slicing, filtering, and avoiding slow loops using vectorized operations.

Practice

Generate a Kaggle/Colab notebook with 15 unique data manipulation and multi-index challenges.

Session 2: Merging & Aggregations

Lecture

Mastering .groupby(), .merge(), .join(), and constructing dynamic summary tables.

Practice

Combine multiple relational tables and build custom cross-feature aggregate statistics.

Session 3: Data Cleaning & Imputation

Lecture

Imputation strategies (mean, median, mode) vs. tracking missing values with explicit Boolean indicator flags.

Practice

Take a heavily corrupted dataset, track missingness percentages, and build a clean processing function.

Day 3: Feature Engineering I — Basic Transformations

Session 1: Categorical Encoding

Lecture

One-Hot Encoding vs. Label Encoding vs. Frequency Encoding. Managing column cardinality.

Practice

Process high-cardinality values on a mock notebook without causing memory issues or data leakages.

Session 2: Numerical Scaling & Distributions

Lecture

Handling skewness, log transforms, Standard Scaling ($z$-score), and MinMax Scaling choices.

Practice

Identify skewed numeric columns, apply transformations, and map distributions using Seaborn plots.

Session 3: Interaction & Binning Features

Lecture

Mathematical combinations of columns (ratios, products) and segmenting continuous metrics into ordinal bins.

Practice

Review the Titanic dataset. Write a function to construct 5 entirely new baseline interaction features.

Day 4: Feature Engineering II — Advanced "Golden" Features

Session 1: Target Encoding & Smoothing

Lecture

Encoding categories based on historic target values. The massive risk of overfitting and how to apply smoothing weight formulas.

Practice

Manually build a Target Encoder with out-of-fold calculations to safeguard against leakage.

Session 2: Group-Level Transform Features

Lecture

Utilizing .transform() in Pandas to create localized context metrics (e.g., individual value vs group mean).

Practice

Build 10 contextual features on a messy transactional transaction dataset using group-level summaries.

Session 3: Feature Selection & Drifting

Lecture

Dropping collinear rows, variance thresholds, and evaluating initial baseline model feature importance scores.

Practice

Prune the bottom 50% of low-performing features from a massive wide dataset without losing baseline accuracy.

Day 5: Validation Architecture & Baselines

Session 1: The Cross-Validation Framework

Lecture

K-Fold vs. Stratified K-Fold vs. Time-Series Splits. Why chasing public leaderboards blindly is a catastrophic trap.

Practice

Write a highly robust, fully reusable Stratified 5-Fold cross-validation loop configuration from scratch.

Session 2: Baseline Model Implementation

Lecture

Scikit-Learn API consistency. Training rapid reference baselines (Ridge, Logistic Regression, Random Forest).

Practice

Plug a baseline classifier into your 5-fold architecture loop and generate an end-to-end output submission file.

Session 3: Leakage Prevention Auditing

Lecture

Common ways data leaks (e.g., scaling datasets globally before splitting, encoding features outside CV folds).

Practice

Audit a broken notebook built by your AI. Locate the data leakage flaws and refactor them to fix errors.

Day 6: Tree-Based Gradient Boosting Dominance

Session 1: Introduction to LightGBM

Lecture

Why LightGBM dominates tabular metrics. Leaf-wise growth, missing values handling, and exceptional compute speed.

Practice

Implement LightGBM inside your CV loop framework. Compare validation metrics directly against your baseline models.

Session 2: Introduction to XGBoost

Lecture

Level-wise growth structures, L1/L2 regularization parameters, and handling structural weights.

Practice

Deploy XGBoost side-by-side on the exact same cross-validation splits to track variations in predictions.

Session 3: Tree Hyperparameter Deep-Dive

Lecture

Structural constraints (max_depth, num_leaves) vs. stochastic adjustments (subsample, colsample_bytree).

Practice

Manually alternate model hyperparameters to visually analyze training vs. validation score deltas.

Day 7: Optimization & Advanced Ensembling

Session 1: Automated Tuning with Optuna

Lecture

Introduction to Bayesian search optimization concepts. Constructing an Optuna objective evaluation function.

Practice

Set up an automated Optuna script sequence to study the parameters of your LightGBM algorithm for 50 trials.

Session 2: Model Blending & Averaging

Lecture

Ensembling principles. Combining low-correlation model structures to smooth prediction errors.

Practice

Extract prediction metrics from your best LightGBM and XGBoost iterations, blending them mathematically ($0.6 \times \text{LGBM} + 0.4 \times \text{XGBoost}$).

Session 3: Metric Optimization & Post-Processing

Lecture

Aligning target models directly with competition metrics. Finding the exact probability cutoff points instead of standard 0.5 rules.

Practice

Construct a custom threshold scanning loop (0.01 to 0.99) to extract the absolute maximum F1-score optimization coordinate.

Day 8: Production Pipelines & Collaborative Git Workflows

Session 1: Modularizing Code into Scripts

Lecture

Migrating away from unorganized notebooks. Architecting standard clean file paths: data_prep.py, models.py, main.py.

Practice

Convert your week's functional features, preprocessing parameters, and loops into clean modular executable scripts.

Session 2: Git Collaboration Workflows

Lecture

Managing branching structures, pull requests, and dealing with fast merge conflicts under hackathon stress.

Practice

Simulate a live sprint: Team Member A pushes preprocessing modifications, Team Member B pulls and incorporates ensembling modules.

Session 3: The "Secret Weapon" Master Template

Lecture

Consolidating your ultimate ready-to-run template notebook for instant deployment.

Practice

Lock in a template that reads a tabular file, calculates missing values, applies K-Fold, and exports predictions in under 30 minutes.

Section 2: Day 9 — The 4-Hour Custom Mock Competition

Rules of Engagement: Pick a historic, unread Kaggle tabular dataset. No external code lookups allowed. Use your Day 7 template pipeline, standard documentation, and team communications exclusively.

00:00 - 00:30

Phase 1: Exploration, EDA, & Strategy Assignment

Read the problem instructions, evaluate shapes, and delegate tasks (e.g., preprocessor setup vs baseline feature brainstorming).

00:30 - 01:15

Phase 2: The Baseline Lock-In

Feed the raw CSV files straight into your pipeline template. Output and submit a clean, functional prediction track immediately to ensure execution path viability.

01:15 - 02:45

Phase 3: The Feature Sprint

Write mathematical, structural, and categorical transformations in parallel streams. Test features individually; drop them ruthlessly if local CV drops.

02:45 - 03:30

Phase 4: Optuna Hyperparameter Optimization

Fire up automated hyperparameter tuning processes for LightGBM and XGBoost tracking the final chosen features.

03:30 - 04:00

Phase 5: Ensemble blending & Submission Polish

Blend final probability sets. Check for structural alignment with sample format configurations. Lock in final sub files.

🔑 The Three Golden Rules for the Hackathon Leaderboard

  • Trust Your Local CV: If your local Cross-Validation framework score improves but the public Kaggle leaderboard fluctuates downward, always trust your CV score. Public boards are prone to statistical noise.
  • Submit Inside Hour One: Never leave submission tasks to the final hours. Lock a basic baseline file into the tracking systems immediately.
  • No Optimization Without Validation: Delete features that sound "clever" but fail to register positive metric improvements in your 5-Fold split validation.
